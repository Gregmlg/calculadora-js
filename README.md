# 🧮 Calculadora JS

Calculadora básica en JavaScript puro.
Proyecto de ejercicio para aprender GitHub.

## Operaciones disponibles

- ➕ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División (con control de división por cero)

## Uso

```bash
node calculadora.js
```

## Versiones

- **v1.0** - Operaciones básicas
